Lesson 1: What is Arduino?

Arduino is an open-source electronics platform used for building interactive projects. It consists of programmable microcontroller boards and easy-to-use software called the Arduino IDE.

Why Arduino?

  • Easy for beginners
  • Low cost
  • Large community support
  • Used in robotics, IoT and automation
  • Supports many sensors and modules

Quick Notes


Arduino = Hardware + Software



Hardware:

- Arduino Board



Software:

- Arduino IDE



Uses:

- Robotics

- Home Automation

- IoT

- Smart Projects

Lesson 2: History of Arduino

Arduino was created in Italy in 2005 to provide students with a simple and affordable platform for learning electronics and programming.

Interesting Facts

  • Started in 2005
  • Created in Italy
  • Open-source platform
  • Millions of users worldwide

Founded: 2005

Lesson 4: Understanding Arduino Pins

Digital Pins

Digital pins can be used as INPUT or OUTPUT. They only understand HIGH (5V) and LOW (0V). 


Digital Pins:

D0 - D13



Examples:

LED

Button

Relay

Buzzer

Analog Pins

Analog pins can read varying voltages from sensors. 


Analog Pins:

A0 - A5



Examples:

LDR

Temperature Sensor

Soil Moisture Sensor

Joystick

PWM Pins

PWM pins simulate analog output. 


PWM Pins:

3, 5, 6, 9, 10, 11



Used For:

LED Brightness

Servo Control

Motor Speed

Lesson 5: First Arduino Program

The most famous Arduino program is Blink. 


void setup() {



pinMode(13, OUTPUT);



}



void loop() {



digitalWrite(13, HIGH);

delay(1000);



digitalWrite(13, LOW);

delay(1000);



}

Explanation


pinMode()

Sets pin mode.



digitalWrite()

Turns pin ON/OFF.



delay()

Waits for given time.
