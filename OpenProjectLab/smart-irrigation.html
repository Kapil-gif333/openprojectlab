Smart Irrigation System

An automatic plant watering system that uses a soil moisture sensor to detect dry soil.

Components Required

  • Arduino Uno
  • Soil Moisture Sensor
  • Relay Module
  • Water Pump
  • Power Supply
  • Jumper Wires

Connections

Component Arduino Pin
Soil Sensor AO A0
Relay IN D8
Relay VCC 5V
Relay GND GND

Arduino Code

int soilPin = A0;
int relayPin = 8;

void setup() {

pinMode(relayPin, OUTPUT);

Serial.begin(9600);

}

void loop() {

int moisture = analogRead(soilPin);

Serial.println(moisture);

if(moisture > 600){

digitalWrite(relayPin, HIGH);

}
else{

digitalWrite(relayPin, LOW);

}

delay(1000);

}
