Line Follower Robot

A robot that automatically follows a black line using IR sensors and DC motors.

Components Required

  • Arduino Uno
  • IR Sensor Module (2 Sensors)
  • L298N Motor Driver
  • 2 DC Gear Motors
  • Robot Chassis
  • Battery Pack
  • Jumper Wires

Connections

Component Arduino Pin
Left IR Sensor D2
Right IR Sensor D3
L298N IN1 D5
L298N IN2 D6
L298N IN3 D9
L298N IN4 D10

Arduino Code

int leftSensor = 2;
int rightSensor = 3;

void setup() {

pinMode(leftSensor, INPUT);
pinMode(rightSensor, INPUT);

Serial.begin(9600);

}

void loop() {

int left = digitalRead(leftSensor);
int right = digitalRead(rightSensor);

if(left == LOW && right == LOW){
Serial.println("Forward");
}
else if(left == LOW){
Serial.println("Turn Left");
}
else if(right == LOW){
Serial.println("Turn Right");
}
else{
Serial.println("Stop");
}

delay(100);

}
