Line Follower Robot

Concept

Robot follows black line using IR sensors.

Code

int leftSensor = A0;
int rightSensor = A1;

void setup(){
pinMode(leftSensor, INPUT);
pinMode(rightSensor, INPUT);
}

void loop(){
int left = analogRead(leftSensor);
int right = analogRead(rightSensor);

if(left < right){
// turn left
}
else{
// turn right
}
}
