Smart Door Lock

A simple password-based door lock using Arduino and a keypad.

Components Required

  • Arduino Uno
  • 4x4 Keypad
  • Servo Motor SG90
  • 16x2 LCD (Optional)
  • Jumper Wires

Connections

Component Arduino Pin
Servo Signal D9
Keypad Rows D2-D5
Keypad Columns D6-D9

Arduino Code

#include 

Servo lockServo;

void setup() {

lockServo.attach(9);

}

void loop() {

lockServo.write(0);
delay(2000);

lockServo.write(90);
delay(2000);

}
