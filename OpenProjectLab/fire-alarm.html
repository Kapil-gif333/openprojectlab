Fire Alarm System

📦 Components

  • Arduino UNO
  • MQ2 Smoke Sensor
  • Buzzer

🔌 Connections

ComponentPin
MQ2 SensorA0
BuzzerD8

💻 Code

int sensor = A0;
int buzzer = 8;

void loop(){
int value = analogRead(sensor);

if(value > 400){
digitalWrite(buzzer,HIGH);
}
else{
digitalWrite(buzzer,LOW);
}
}
