Robot Arm Project

A simple servo-based robot arm controlled by Arduino.

Components Required

  • Arduino Uno
  • 2 x SG90 Servo Motors
  • Robot Arm Kit
  • Jumper Wires
  • External Power Supply

Connections

Component Arduino Pin
Base Servo D9
Arm Servo D10
Servo VCC 5V
Servo GND GND

Arduino Code

#include 

Servo baseServo;
Servo armServo;

void setup() {

baseServo.attach(9);
armServo.attach(10);

}

void loop() {

baseServo.write(0);
armServo.write(0);

delay(1000);

baseServo.write(90);
armServo.write(90);

delay(1000);

baseServo.write(180);
armServo.write(180);

delay(1000);

}
