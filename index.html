Fire Alarm System

A simple fire alarm system using a flame sensor and buzzer to detect fire.

Components Required

  • Arduino Uno
  • Flame Sensor Module
  • Piezo Buzzer
  • LED
  • 220Ω Resistor
  • Jumper Wires

Connections

Component Arduino Pin
Flame Sensor OUT D2
Buzzer D8
LED D13

Arduino Code

int flameSensor = 2;
int buzzer = 8;
int led = 13;

void setup() {

pinMode(flameSensor, INPUT);
pinMode(buzzer, OUTPUT);
pinMode(led, OUTPUT);

}

void loop() {

int fire = digitalRead(flameSensor);

if(fire == LOW){

digitalWrite(buzzer, HIGH);
digitalWrite(led, HIGH);

}
else{

digitalWrite(buzzer, LOW);
digitalWrite(led, LOW);

}

delay(100);

}
