Home Automation System

Control home appliances using Arduino and relay modules.

Components Required

  • Arduino Uno
  • 4 Channel Relay Module
  • LEDs (for demonstration)
  • Jumper Wires
  • Power Supply

Connections

Component Arduino Pin
Relay 1 D2
Relay 2 D3
Relay 3 D4
Relay 4 D5

Arduino Code

int relay1 = 2;

void setup() {

pinMode(relay1, OUTPUT);

}

void loop() {

digitalWrite(relay1, HIGH);
delay(2000);

digitalWrite(relay1, LOW);
delay(2000);

}
